# Typography

Our typography system is implemented as **Tailwind CSS utility classes** that use design tokens for font size, weight, line height, and letter spacing.

Use classes like `.text-t1`, `.text-t2`, `.text-t3`, etc., to apply these typography styles throughout your app.

## Typography Scale

Our typography system supports a responsive and accessible interface. Each text token uses custom properties (CSS variables) that define:

- Font size
- Line height
- Letter spacing
- Font weight
